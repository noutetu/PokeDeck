PokeDeck
PokeDeck は、ポケモンカードゲームのデッキ管理アプリケーションです。カードコレクションの管理やデッキ構築が簡単に行えます。



🔧 主な機能
📇 カード管理
所有カードの一覧表示と検索

カード詳細情報の閲覧（HP、タイプ、技、特性など）

カードのフィルタリングとソート（タイプ、進化段階、パック別など）

🧩 デッキ構築
複数のデッキの作成と保存

ドラッグ&ドロップによる直感的なデッキ編集

デッキ統計情報の表示（タイプ分布、コスト曲線など）

エネルギーの自動選択機能（デッキ内のポケモンタイプに基づく）

🔍 検索機能
カード名、タイプ、パック名などによる高度な検索

タグベースのフィルタリング（回復、ドローなど）

🏗️ アーキテクチャ
本アプリは MVP（Model-View-Presenter）アーキテクチャ に基づいて設計されています。

モデル層
CardModel：カード情報を保持するデータクラス

DeckModel：デッキ情報とカードリストを管理

CardDataManager：カードデータの読み込みと管理を担当

ビュー層
CardView：個別カードの表示を担当

DeckView：デッキの表示と操作インターフェースを提供

SearchView：検索インターフェースを管理

プレゼンター層
ビューとモデル間の橋渡し

ユーザー操作に応じたロジックの実行

データをビジュアル表示用に加工
